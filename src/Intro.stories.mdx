# Get started

Roa Design System is a reusable component library that helps Roa contributors build UIs faster. The goal is to make building durable UIs more productive and satisfying.

## Install

Roa components are written in React, and its stories are written in Component Story Format. It requires Storybook version 6.1.20 and up.

Add SDS to your project.

```
npm install --save @roa/ui
```

## Use

Import components you want into your UI

```tsx
import { Button } from "@roa/ui";
```

and use them like so

```tsx
const Example = () => (
  <Button text="Hello Roa 👋" onClick={() => console.log("Hello Roa 👍")} />
);

export default Example;
```
